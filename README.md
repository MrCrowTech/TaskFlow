# TaskFlow

A simple to-do app where you can add tasks, remove them, and switch between
states like pending, in progress, and done. Built with plain HTML, CSS, and
JavaScript, roughly a year into teaching myself web development.

## What It Does

- Add new tasks with a title and short description
- Move tasks through different states (pending, in progress, done)
- Delete tasks once you're done with them
- See everything in a clean column-based layout

Nothing fancy, just the core loop of a task app done from scratch.

## What I Practiced

- Working with the DOM and event listeners
- Storing tasks in localStorage so they survive a refresh
- Building a kanban-style layout with CSS Grid
- Keeping state and UI in sync by hand
- Thinking through small UX details like empty states and confirmations

## Run It Locally

1. Clone the repo
2. Open `index.html` in your browser
3. Start adding tasks

No installs, no dependencies, no server needed.
